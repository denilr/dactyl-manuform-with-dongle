Это сборка прошивки zmk для dactyl manuform.
С прошивкой контроллеров была проблема в нерабочем соединении половинок между собой или с донглом, как выяснилось файлы конфигов .defconf, dactyl.conf не работают в соответствии с руководством zmk https://zmk.dev/
**Конфиги работают только в файлах dactyl_left.conf, dactyl_right.conf, dactyl_dongle.conf**, т.е. в конфигах каждого отдельного контроллера.
Если необходимо изменить конфиги, то менять только перечисленные 3 файла, в других местах изменения не работают. 

Причина такого поведения конфигов не ясна, на других сборках клавиатур с такими же контроллерами (китайские nice!nano на nrf52840) таких проблем не возникало(ет)
